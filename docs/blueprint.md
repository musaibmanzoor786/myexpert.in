# **App Name**: MyExpert

## Core Features:

- Authentication: Email/Password Authentication with role selection (Customer/Expert).
- Expert Directory: Browse experts based on service type and location.
- Expert Profiles: Display expert profiles with detailed information such as services offered, rates, and reviews.
- Reviews and Ratings: Provide a review system with ratings.
- Contact Experts: Enable users to directly contact experts through in-app messaging or call functionality. It is the experts decision as to which contact methods they would like to enable.
- AI Profile Summaries: Use an LLM tool to create a short summary for each expert profile based on uploaded documents.

## Style Guidelines:

- Primary color: Teal (#008080) to convey trust and reliability.
- Background color: Light teal (#E0F8F7) for a clean and calming background.
- Accent color: Yellow-gold (#D4AF37) to highlight key actions and information.
- Body and headline font: 'Inter' (sans-serif) for a modern, objective, and neutral user interface.
- Use consistent, professional icons representing service categories and actions.
- Implement a clean, card-based layout for expert listings.
- Subtle transitions and animations for a smooth user experience.
